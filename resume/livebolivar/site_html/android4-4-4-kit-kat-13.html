LIVEBOLIVAR.COM ON ANDROID 4.4.4 DEVICES

  1. ENABLE FLASH IN THE DOLPHIN BROWSER (HAVEN'T ENABLED FLASH IN CHROME 55 AS OF 010217)
    1. DOWNLOAD ARCHIVED ADOBE FLASH PLAYER
      1. Flash Player 11.1 for Android 4.0 (11.1.115.81)
    2. ENABLE "UNKNOWN SOURCES"
      1. FROM ANDROID SETTINGS:
        1. SECURITY - UNKNOWN SOURCES - PLACE CHECKMARK
    3. INSTALL FROM ANDROID [FILE MANAGER - APPLICATIONS] IF YOUR BROWSER WILL NOT INSTALL IT. CHROME WOULD NOT INSTALL IT EVEN AFTER RESTARTING THE PHONE.
    4. INSTALL AND OPEN THE DOLPHIN JETPACK FROM THE GOOGLE PLAY STORE
    5. FROM THE DOLPHIN BROWSER:
      1. DOLPHIN - SETTINGS:
        1. BASIC
          1. DOLPHIN JETPACK - ON
        2. WEB CONTENT
          1. FLASH PLAYER - ALWAYS ON
  2. CHECK TO SEE IF YOUR WORK WORKED. THE MOON PHASE IMAGE IS FLASH CONTENT.
  1. ENABLE DEVELOPER OPTIONS ON YOUR ANDROID 4.4.4 DEVICE
    1. Open the Android Settings - System - About Phone menu. Near the bottom, you should see the "SW Version" header. Tap this area about 10 times and Developer Options will be unhidden. Once you've done this Developer Options should be at Settings - System - Developer Options.
